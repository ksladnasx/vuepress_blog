对基础 SSE 中流的处理的解读

解析

这是一段实现前端流式接收和处理 SSE（Server-Sent Events）数据的代码，它没有使用浏览器原生的EventSource，而是采用fetch配合流式读取来获取并处理数据，从而实现了逐字输出的效果。下面我们来逐部分解析。

  • if (!response.ok || !response.body) { ... }：检查响应是否成功（状态码 200-299）以及响应体（body）是否存在。如果任一条件不满足，则抛出错误。
  • const reader = response.body.getReader();：从响应体中获取一个可读流读取器（ReadableStreamDefaultReader）。这个读取器允许你逐步读取响应体中的数据块，而不是一次性加载全部内容。调用getReader()方法会锁定这个流，使其不能被其他读取器使用。
  • const decoder = new TextDecoder();：创建一个TextDecoder对象，用于将接收到的二进制数据块（通常是Uint8Array）解码成字符串。它默认使用 UTF-8 编码。

  • if (done) { ... }：如果流已结束，则更新状态并退出函数。
  • const chunk = decoder.decode(value, { stream: true });：使用TextDecoder的decode方法将二进制数据块解码成字符串。关键点在于{ stream: true }这个选项：它告诉解码器数据是分块传输的，可能包含不完整的字符（比如多字节字符如中文被分割在不同的数据块中），解码器会保留这些不完整的字节序列，等待下一次decode调用时再组合成完整的字符，从而避免乱码。
  • const lines = chunk.split('\n');：SSE 协议中，每条消息以换行符分隔。这里将解码后的字符串按行分割，便于逐行处理。

💎 核心机制总结

这段代码的关键在于使用fetch的流式 API 逐步读取和处理 SSE 数据，而非一次性加载。它通过getReader()获取数据流读取器，循环调用read()方法读取数据块，用TextDecoder解码（注意{ stream: true }选项），并按照 SSE 协议解析数据行，最终实现逐字更新的视觉效果。

完整代码

const startStream = async () => {
  if (isStreaming.value) return;

  try {
    outputText.value = "";
    isStreaming.value = true;

    const response = await fetch("http://localhost:3000/stream-text");
    if (!response.ok || !response.body) {
      throw new Error("流请求失败");
    }

    //将传来 的二进制数据流转换为文本
    const reader = response.body.getReader();
    const decoder = new TextDecoder();

    const processStream = ({ done, value }) => {
      if (done) {
        isStreaming.value = false;
        return;
      }

      const chunk = decoder.decode(value, { stream: true });
      const lines = chunk.split("\n");

      for (const line of lines) {
        if (line.startsWith("data: ")) {
          try {
            const data = JSON.parse(line.substring(6));

            if (data.done) {
              isStreaming.value = false;
              return;
            }

            if (data.char === "\n") {
              outputText.value += "\n";
            } else {
              outputText.value += data.char;
            }

            // 自动滚动到底部
            nextTick(() => {
              if (outputElement.value) {
                outputElement.value.scrollTop =
                  outputElement.value.scrollHeight;
              }
            });
          } catch (e) {
            console.error("解析消息错误:", e);
          }
        }
      }

      return reader.read().then(processStream);
    };
    reader.read().then(processStream);
  } catch (error) {
    console.error("流式请求错误:", error);
    isStreaming.value = false;
  }
};
